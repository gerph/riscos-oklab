# OKLab colour space for RISC OS

This is a port to RISC OS of the OKLab colour space library.
OKLab colour space is designed to improve uniformity in hue and
lightness of colours.

See the [Wikipedia page](https://en.wikipedia.org/wiki/Oklab_color_space) on the OKLab colour space,
and the [main project page](https://bottosson.github.io/posts/oklab/).
